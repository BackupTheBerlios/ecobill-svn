Hier werden ihre Fragen zum Nachrichtenbereich beantwortet

1. Erstellen einer neuen Nachricht:

Durch drücken des Button werden die Textfelder Absender, Betreff und Nachricht gelöscht und man kann so eine neue Nachricht mit Absender und Betreff eingeben.

2. Abspeichern einer neuen Nachricht:

Geben sie einen Absender, Betreff und eine Nachricht ein und drücken sie den Button schon wird die Nachricht im Übersichtsbaum unter ihrem Absendernamen und Betreff abgelegt.

3. Löschen einer Nachricht:

Durch drücken des Button wird die durch Sie im Übersichtsbaum ausgewählte Nachricht gelöscht. Falls dies die letzte Nachricht eines Benutzers war wird dieser auch gelöscht.

4. Aktualisieren:

Durch drücken des Button wird der Übersichtsbaum und die Textfelder aktualisiert.
