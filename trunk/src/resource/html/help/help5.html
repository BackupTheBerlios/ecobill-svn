Hier werden ihre Fragen zum Rechnungsbereich beantwortet

1. Erstellen einer neuen Rechnung:

Durch drücken des Button werden die Textfelder gelöscht und man kann so einen neuen Kunden eingeben.

2. Abspeichern einer neuen Rechnung:

Geben sie alles notwendige ein und drücken sie den Button schon wird der Kunde in der Tabelle unten abgelegt und steht ihnen zur Verfügung.

3. Löschen einer Rechnung:

Durch drücken des Button wird der durch Sie in der Tabelle ausgewählte Kunde gelöscht.

4. Aktualisieren:

Durch drücken des Button wird die Tabelle und die Textfelder aktualisiert.
